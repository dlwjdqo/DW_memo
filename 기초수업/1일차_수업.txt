### 면접 문제

* 형상관리 툴 (소프트웨어 버전 관리 툴)
종류 : Git, SVN

SVN Git 차이점 :
=> SVN 은 중앙 *서버가 존재, git 은 존재하지 않음
    *server : (서비스를 제공해주는 주체)
     -> 서비스를 제공해주면 다 서버가 됨.
-> git (개인은무료, 그룹화시 인원 추가대비 비용발생)
    스타트업 및 기업들에서 많이 사용함
ex) 두 사람이 따로 작업해도 서로 공유 및 보관

-> SVN
    공공기관, 연구소, 국가기관에서 많이 사용함
ex) 공공기관 사용이유
    ->두 사람이 따로 작업해도 SVN에 한번에 보관

IP 인터넷을 사용할 수 있는 집 주소 (고유번호 x 집 주소처럼 변경 가능)

호스트 번호는 맨뒷자리며 집 상세주소 개념임
