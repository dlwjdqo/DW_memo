데이터 베이스 -> 테이블 -> 데이터

SELECT
    컬럼이름
FROM 테이블
WHERE 조건식 (연산자)
GROUP BY
HAVING 집계함수, 그룹핑된 컬럼들 조건
ORDER BY 컬럼이름 ASC
LIMIT

쿼리순서
FROM -> WHERE -> GROUP BY -> HAVING -> SELECT
-> ORDER BY -> LIMIT
