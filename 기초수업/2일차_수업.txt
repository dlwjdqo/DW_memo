***git 명령어

git init -> git 을 사용할 수 있게 해주는 초기세팅 명령어 (중앙서버 개념)
git clone -> github에있는 폴더를 다운
git status -> 현재 git 상태확인
git add -> 커밋전 이루워지는 행동 ( 파일 및 폴더 추가시 사용 내용만 추가시 사용안함 )
*git commit -> 내용만 수정 후 저장시에 사용
*git push -> github 에 저장시 사용 ( 공통작업시 push전 통보 )
*git pull -> github에 업로드한 내용 동기화 ( 제일먼저 입력 후 작업 시작 )
fix -> 내용 수정시 입력 
feat -> 추가시 입력
