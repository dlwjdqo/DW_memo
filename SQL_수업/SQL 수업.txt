### 수업

EXISTS : 서브쿼리가 TRUE면 메인쿼리 실행
        FALSE면 실행 안함 (update시 사용함)

NOT EXISTS : 서브쿼리가 FALSE면 실행

ex)
    select
        *
    FROM emp
    WHERE EXISTS (
        select *
        FROM emp WHERE
        job = 'manager'
    )

사용예시
--카카오뱅크
    update 백업실행
    set 백업날짜 = '220222'
    WHERE EXISTS
    (
        select * from 입출금내역
    )

PK, FK 정의
 PRIMARY KEY(PK)
 -> 중복 허용 X
 -> 자동으로 인덱스가 붙음.

 FOREIGN KEY(FK)
 -> 중복 허용 O -> 중복 허용 X (unique key)
 -> 인덱스 없음
 -> index를 생성해야 함.
 문법)
 CREATE INDEX <인덱스명>
 ON <테이블명> ( 칼럼명1, 칼럼명2, ...);
